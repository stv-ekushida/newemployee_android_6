# newemployee_android_6
ConstraintLayout

## デザインモード
制約ハンドルを使って制約を設定する

| パターン | 説明 |
|---|---|
|Wrap Content | wap_content |
|Fixed | 具体的な数値 |
|Match Constraints | 設定値は0dp match_parent |

### ベースライン
レベルと入力欄の位置を揃える

①右クリック　→　[Show Baseline]
②ドラッグ＆ドロップ

### ガイドライン
ビューの左の位置を揃える

①メニュー　→　[Add Vertical Guidline]
②ガイドラインの位置を変更する

### チェイン機能
ビューを均等に配置する

①右クリック　→ [Chains] →　[Create Horizontal Chain]
